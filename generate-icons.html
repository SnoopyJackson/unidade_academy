BJJ Foundation - App Icon Generator

Generate PNG icons for Android & iOS from the master SVG

How to use these icons

  1. Click "Generate All Icons" to render PNGs from the SVG icon.
  2. Download each icon individually or click "Download All Icons".
  3. Place PNG files in an icons/ folder in your project root.
  4. Android (PWA): The manifest.json is already configured. Add to home screen from Chrome.
  5. iOS: The apple-touch-icon meta tags in index.html handle the home screen icon.

Android Icon Sizes

  1. 48x48 — mdpi (baseline)
  2. 72x72 — hdpi
  3. 96x96 — xhdpi
  4. 144x144 — xxhdpi
  5. 192x192 — xxxhdpi / PWA required
  6. 512x512 — Google Play Store / PWA splash
